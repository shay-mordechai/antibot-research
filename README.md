# Agentic Bot Research Lab

This repository acts as a central hub for my research on **Agentic AI behavior**, bot evasion techniques, and the development of defensive countermeasures. 

## Research Modules

- [**`honeypot-experiment/`**](./honeypot-experiment/)
  Empirical research into how Agentic bots interact with static vs. behavioral defenses, including TLS/JA3 analysis and DOM interaction patterns.

- [**`agentic-dast-scanner/`**](./agentic-dast-scanner/)
  An autonomous DAST scanner implementation using vision-based navigation to explore and test application vulnerabilities.

- [**`intel/`**](./intel/)
  High-level threat intelligence reports and synthesized threat models regarding Agentic AI.

---
*Research conducted in a controlled environment for security verification purposes.*
